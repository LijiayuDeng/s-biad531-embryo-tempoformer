# EmbryoTempoFormer (S-BIAD531)

Code and analysis scripts for time-lapse developmental time/tempo estimation in zebrafish embryos.

Data source: BioImage Archive (EMBL-EBI), accession S-BIAD531
https://www.ebi.ac.uk/bioimage-archive/galleries/S-BIAD531.html

## Quickstart (skeleton)
1) Copy `.env.example` to `.env` and edit paths.
2) Use `src/EmbryoTempoFormer.py` CLI: preprocess / train / infer.
